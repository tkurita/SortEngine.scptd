set_comparator

set_comparator(a_comparator)

comparator script を設定します。

comparator script は以下の仕様を満たしている必要があります。

  • 二つの positional parameter を持つ do handler を実装している。
  • do handler は次の値を返す。
    • 1 : 第一引数 > 第二引数
    • 0 : 第一引数 = 第二引数
    • -1 : 第一引数 < 第二引数
Parameters
  • a_comparator (script object)
Result

script object : me
